# WeatherApp

A modern Android Weather Application built with Jetpack Compose. This project demonstrates how to fetch and display weather data from the [Open Meteo API](https://open-meteo.com/en).

## Overview

In this project, we make use of the Open Meteo API to get real-time weather data and nicely display it in an Android app with a clean UI.

## Features

- **Real-time Weather Data**: Fetch current weather conditions and forecasts.
- **Location Services**: Automatically detect the user's location to provide local weather info.
- **Modern UI**: Built entirely using Jetpack Compose for a responsive and fluid user experience.
- **Persistent Storage**: Uses DataStore for saving user preferences.
- **Offline Support**: (Optional) Integrated Room database for caching weather data.

## Tech Stack & Libraries

- **Kotlin**: Primary programming language.
- **Jetpack Compose**: Modern toolkit for building native UI.
- **Hilt**: Dependency injection library for Android.
- **Retrofit & OkHttp**: For network requests and API communication.
- **Coil**: Image loading library for Android.
- **Jetpack DataStore**: Modern data storage solution for small datasets and preferences.
- **Google Play Services Location**: To access device location information.

## API Reference

This app uses the **Open Meteo API**. 
- Website: [https://open-meteo.com/en](https://open-meteo.com/en)
- No API key is required for the free tier, making it easy to get started!
